# Manifest components table

1. Save your full manifest JSON as **`manifest.json`** in this folder (paste the entire JSON object you have).
2. Run:
   ```bash
   python3 parse_manifest.py
   ```
3. Open **`manifest-components-table.md`** for the generated table.

The table columns are: **Component ID**, **Extension Type**, **Display Name**, **Selector**, **Description**, **Archetype**.
